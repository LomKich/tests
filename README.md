# TestSolver 🎓

Android-приложение, которое автоматически читает текст теста на экране и подставляет правильные ответы.  
Работает через **Accessibility Service** — никакого API ключа, полностью бесплатно.

## Скачать APK

1. Перейди на вкладку **Actions** → последний workflow → **TestSolver-debug** в разделе Artifacts.

## Как собрать самому

```bash
git clone https://github.com/ВАШ_АККАУНТ/TestSolver.git
cd TestSolver
# Android Studio → Open → выбрать папку
# Run → Build APK(s)
```

Или просто сделай push — GitHub Actions соберёт APK автоматически.

## Как использовать

1. Установить APK на телефон
2. Открыть приложение
3. Нажать **"Открыть настройки доступности"** → найти `TestSolver` → включить
4. Нажать **"Разрешить наложение окон"** → включить
5. Открыть Chrome → зайти на тест
6. Внизу экрана появится панель с ответом ✅

## Принцип работы

- `AnswerDatabase.java` — загружает `answers.json` с базой ответов
- `TestAccessibilityService.java` — читает текст на экране, ищет совпадение, вставляет ответ / кликает нужный вариант
- `answers.json` — база из ~50 вопросов теста по Материаловедению МПд,МПл-2-24

## Добавить новые ответы

Отредактируй `app/src/main/assets/answers.json`:

```json
{
  "num": 99,
  "keys": ["ключевые слова из вопроса"],
  "type": "text",
  "answer": "ответ"
}
```

Типы: `text` (поле ввода), `radio` (один вариант), `checkbox` (несколько), `match` (соответствие).
